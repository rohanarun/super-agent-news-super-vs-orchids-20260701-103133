Security, realism, and computer access

What recent reporting showed

The BBC reported a zero‑click vulnerability in Orchids that allowed a researcher to access and modify a reporter’s computer via the platform. Experts warned that deep, unsandboxed computer access creates a new class of security risk.

Source: BBC

Why this matters for agents

Once agents can operate computers, mistakes and exploits scale quickly. Security researchers note that users are often advised to run such tools on disposable machines — advice most non‑technical users will not follow.

Source: Vibe Graveyard

Super’s philosophy
